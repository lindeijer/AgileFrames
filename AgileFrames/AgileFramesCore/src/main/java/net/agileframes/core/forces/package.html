Contains the core classes of the FORCES-layer of AgileFrames.

What is FORCES?
FORCES stands for Function Oriented Resource Control and Engineering System. This is the execution layer, the lowest layer that takes care of the real operations on the machines, such as the steering of the wheels. FORCES is a generic system to program Machines from the viewpoint of functionality.
An example is the functional programming of vehicles.

The set-up of FORCES
FORCES operates in the context of a generic system for Traffic Control, called TRACES.
The elementary machine action, called manoeuvre, realises a trajectory which is specified as a function of an evolution-variable.
To co-ordinate manoeuvring with other actors, the speed of the evolution of a manoeuvre can be modified (but not reversed).

Special attention is given to the interface of the physical movement in the continuous time domain and the interaction with Traffic Control in the event-time domain.
