Contains the core classes of the SERVICES-layer of AgileFrames.

What is SERVICES?
SERVICES stands for the Service Coordination and Engineering System. This is the service layer, the highest layer that deals with the client-interface, planning and scheduling. Actors can both execute and demand for services. SERVICES relates to client-interfacing and contracting. Under SERVICES a (so called) server ‘does the job’ in the form of the execution of a basic act.
